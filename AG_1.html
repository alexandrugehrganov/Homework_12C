Disciplinele studiate in clasa a XII-a

Matematica
- este in general definita ca stiinta ce studiaza relatiile cantitative, modelele de structura, de schimbare si de spatiu. In sens modern, matematica este investigarea structurilor abstracte definite in mod axiomatic folosind logica formala.
Limba romana
- este o limba indo-europeana, din grupul italic si din subgrupul oriental al limbilor romanice.
Limba engleza
- este o limba vest germanica avandu-si originile in Anglia, si care este in prezent limba materna pentru majoritatea locuitorilor Australiei, Canadei, Commonwealth-ului Caraibean, Irlandei, Noii Zeelande, Regatului Unit si Statelor Unite ale Americii.
Informatica
- este stiin»õa processrii sistematice a informatiei, in special a procesarii cu ajutorul calculatoarelor.
Istoria
- este stiinta care studiaza evolutia societatii omenesti din trecut pana in prezent.
Geografia
- este stiinta care studiaza relieful, terenurile, trasaturile, locuitorii si fenomenele Pamantului.
Fizica
- este stiinta care studiaza proprietatile si structura materiei, formele de miscare ale acesteia, precum si transformarile lor reciproce.
Chimia
- este reprezinta una dintre ramurile stiintelor naturale al carei obiect de studiu il constituie compozitia, structura, proprietatile si schimbarea materiei.
Biologia
- este stiinta naturala care se ocupa cu studiul vietii si al tuturor organismelor vii.
Educatia civica
- este discipina al carei scop este formarea unui cetatean activ, informat si responsabil, pregatit pentru a trai si a actiona intr-o societate democratica.
Educatia fizica
- este o activitate umana de practicare sistematica si constienta a exercitiilor fizice, desfasurata sub forma de proces instructiv - educativ bilateral, in vederea realizarii unor obiective bine precizate.
